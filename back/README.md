## Requisitos de entorno

Este proyecto debe ejecutarse con Python 3.13.

La combinación actual de FastAPI/Pydantic del entorno falla en Python 3.14 beta al arrancar el servidor, antes de entrar en el código de la aplicación.

Si recreas el entorno, usá el intérprete de `C:\Python313\python.exe` para volver a crear `.venv`.
